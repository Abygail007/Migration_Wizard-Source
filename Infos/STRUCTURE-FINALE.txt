WORKFLOW SIMPLIFIE
------------------

1. DEVELOPPER
   - Editer les fichiers dans source/src/
   - Tester la syntaxe PowerShell

2. COMPILER
   cd C:\Users\jmthomas\Documents\Creation\MigrationWizard\source
   .\Tools\Build-PortableExe.ps1 -Version "1.0.X.X" -IncludeRZGet

   → L'EXE est automatiquement genere dans MigrationWizard-Exe\

3. TESTER
   - Lancer MigrationWizard-Exe\MigrationWizard.exe
   - Tester export + import
   - Verifier les logs dans %USERPROFILE%\MigrationWizard\Logs\

4. COMMIT SOURCES (depot prive)
   cd C:\Users\jmthomas\Documents\Creation\MigrationWizard\source
   git add -A
   git commit -m "Description des changements"
   git push -u source main

5. PUBLISH EXE (depot public)
   cd C:\Users\jmthomas\Documents\Creation\MigrationWizard\MigrationWizard-Exe
   git add MigrationWizard.exe
   git commit -m "Release v1.0.X.X"
   git push -u origin main -f
